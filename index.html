01 Signal

Find What's Outperforming

Virlo creates a "comet" — a saved niche config that tracks your topic space. Every week it pulls outlier videos. Not the biggest creators. Relative outperformers. Videos that beat the creator's OWN baseline.

COMET CONFIGURATION ───────────────────────────── Niche: Fitness & Wellness Keywords: morning routine, workout tips, meal prep Platforms: TikTok, IG Reels, YouTube Shorts Min Views: 10,000 Freshness: Last 7 days ───────────────────────────── Status: ● Active — pulling weekly
02 Content DNA

Break Down Why Winners Work

Each outlier gets broken into 7 dimensions. Then extract the "portable brick" — the structural mechanic that works in ANY niche without copying the original.

THE BUILD Copy-Paste Prompts

The Exact Prompts
Matt Used

Built around 4 pillars: Goal · Process · Tools · Knowledge. These are the actual prompts from the Notion doc.

Prompt 0

Define The Big Idea

Before you build anything, tell the agent what you actually want. Work backwards from the output.

I want to build a weekly content system in OpenClaw. Big idea: Every Monday I want 10 ready-to-post content concepts in my voice, each traced back to a real outlier in my niche that already proved the mechanic works. Work backwards from that output and give me the simplest modular workflow. For each stage, tell me: - the goal - the process - the tools - the knowledge or rules needed Keep it simple and practical. I want a working system I can run first, then improve over time.
Stage 1 — Signal

Create Your Comet

Set up the signal layer. Track what's outperforming in your niche. Focus on relative outperformers — not just the biggest creators.

I want to set up the signal layer for this content system. Goal: - track what is outperforming in my niche Process: - create the niche configuration - save it for reuse - pull recent outliers Tool: - Virlo - Virlo API docs: https://dev.virlo.ai/docs Knowledge: - focus on relative outperformers, not just biggest creators - I care about videos that beat the creator's own baseline My niche is: [INSERT NICHE DESCRIPTION]
Stage 2 — Content DNA

Break Down a Winner

Take one outlier URL from the Virlo output and extract the reusable mechanic. Seven dimensions. One portable brick.

Take one outlier URL from the Virlo output and run a Content DNA analysis on it. Goal: - explain why this specific winner worked - turn that explanation into a reusable mechanic Process: - use ScrapeCreators to fetch the metadata and assets - send that material to google/gemini-3-flash-preview through OpenRouter - break the video into structured dimensions - identify the reusable mechanic Break it down into: 1. Topic 2. Angle 3. Hook structure 4. Story structure 5. Visual format 6. Key visuals 7. Audio Then tell me: - the primary performance driver - the portable brick - why it likely outperformed the creator's baseline
Stage 3 — The Forge

Generate Your Concept

★ Most Important Prompt

This is the creative layer. Take the portable brick and generate a concept that sounds like the creator, not like AI. Show the lineage from source to output.

Use the Content DNA output from that winning video and generate one content concept for my niche. Goal: - turn the winning mechanic into something usable for my brand Process: - keep the transferable mechanic - remove niche-specific surface details - adapt it to my audience and voice Important: - if my voice file exists, use it - if it does not exist, clearly label the output as demo mode - do not give me generic slop - show the lineage from source mechanic to final concept - do not invent fake specifics without labeling them - force this test to use one source only Return: - source inspiration - portable brick - why it transfers - brand voice translation - final hook - content structure - platform - confidence - use this when
Stage 4 — Monday Drop

Run the Full Pipeline

End-to-end test. Proves the pipeline works as one connected system before you hand it to a client.

Now run the weekly workflow locally as an end-to-end test. Goal: - prove the pipeline works as one system Process: - use the current niche config - run the default workflow - save outputs to the workspace Default workflow: Virlo → Content DNA Batch → Pattern Report → The Forge → Monday Drop Run the local workflow with no delivery. Then show me: - what ran - where the outputs were saved - what to inspect if a later user wants to debug or improve
